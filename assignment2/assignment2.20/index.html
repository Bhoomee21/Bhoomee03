Custom CSS Framework Documentation

Grid System

The grid system uses flexbox to create responsive layouts.

                
                    <div class="container">
                        <div class="row">
                            <div class="col-6">
                                Content here
                            </div>
                            <div class="col-6">
                                Content here
                            </div>
                        </div>
                    </div>

Buttons

Buttons with primary and secondary styles.

                
                    <a href="#" class="button button-primary">Primary Button</a>
                    <a href="#" class="button button-secondary">Secondary Button</a>

Cards

Card component for displaying content.

                
                    <div class="card">
                        <h3 class="card-title">Card Title</h3>
                        <p class="card-text">Card content goes here.</p>
                    </div>

Forms

Form components like input fields and textareas.

                
                    <div class="form-group">
                        <label for="username" class="form-label">Username:</label>
                        <input type="text" id="username" class="input-field" placeholder="Enter your username" required>
                    </div>

                    <div class="form-group">
                        <label for="message" class="form-label">Message:</label>
                        <textarea id="message" class="textarea-field" placeholder="Enter your message" rows="4" required></textarea>
                    </div>

Utility Classes

Utility classes for quick styling adjustments.

                
                    <div class="text-center">
                        Centered text
                    </div>

                    <div class="margin-top-20">
                        Content with 20px top margin
                    </div>

                    <div class="padding-20">
                        Content with 20px padding
                    </div>
