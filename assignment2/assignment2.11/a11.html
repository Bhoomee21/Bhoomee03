CSS Positioning Techniques

Static Positioning

This is the default positioning for all elements. Elements are positioned according to the normal flow of the document.

Relative Positioning

Relative positioning moves an element relative to its normal position. It does not remove the element from the document flow.

Absolute Positioning

Absolute positioning removes the element from the normal document flow and positions it relative to its nearest positioned ancestor.

Fixed Positioning

Fixed positioning removes the element from the normal document flow and positions it relative to the viewport. It stays fixed in its position even when the page is scrolled.

Sticky Positioning

Sticky positioning is a hybrid of relative and fixed positioning. It is treated as relative positioning until a specified scroll point is reached, then it is treated as fixed positioning.
